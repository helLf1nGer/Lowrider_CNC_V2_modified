# Lowrider_CNC_V2_modified
A modified version of Lowrider MPCNC

Courtesy to the Creator of the Lowrider MPCNC https://v1engineering.com

Improved current design by adding a second x-axis NEMA 17 motor. The build consists of 6 motors, 2 for each axis, this improved the precision to 0.2mm (soft wood only) 
The build includes:
-grbl 1.1 control
-rigid steel/wood base
-dust shoe
